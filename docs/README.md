# Hospital Management System — Documentation

Professional step-by-step guides for every module.

## In the admin panel

Open **Help** from the top navbar → `/admin/help`

- Browse modules by category
- Search modules by name
- Switch **English** / **বাংলা**
- Each module has numbered steps, field tables, workflows, tips, and FAQ
